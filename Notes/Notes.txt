CSS (Cascading style sheet)

- Designing your page 

3 types of stylesheet:

1.Inline stylesheet : styles inside the html element 

<body style="background-color: aqua;">
    <h1 style="color: red;">Welcome USER</h1>


2.Internal stylesheet : Styles defined on the <style> tag on head section of your html page

  <style>
        table{
            border:10px solid green;
        }
        th
        {
            color:red;
        }
        td{
            background-color: yellow;
        }
    </style>

3. External Stylesheet : Seperate .css file to hold the styles of all the web pages


CSS Selector : What basis or criteria the styles are applied to the html page

1.Element Selector : Apply the styles through html element 

body{
    background-color: burlywood;
}

p{
    color:green;
    text-align: center;
}

 <p>This is element selector styling</p>


2.id Selector : # symbol.Referenced through the id only #p1

#para1{
    color:blue;
    text-align: right;
}

<p id="para1">This is an example for id selector</p>
    <p id="para1">This is my third paragraph</p>

3.class selector : represented (.) symbol 

.mystyle{
    font-size: xx-large;
    font-family: 'Gill Sans', 'Gill Sans MT', Calibri, 'Trebuchet MS', sans-serif;
    color: deeppink;
}

p.mystyle1{
    color: white;;
}


4.Universal selector : (*) all html elments on the page

*{
    color:black;
    text-align: left;
    background-color: aqua;
}
